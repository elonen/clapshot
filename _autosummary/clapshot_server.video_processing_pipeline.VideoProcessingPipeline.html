clapshot_server.video_processing_pipeline.VideoProcessingPipeline¶

class VideoProcessingPipeline(db_file, data_dir, mpm, max_workers=0)[source]¶

Bases: object

Message queue manager.

Create a dispatcher that manages the message queues between the various worker pools.

Parameters

  • db_file (str) – Path to the SQLite database file

  • data_dir (Path) – Path to the data directory (incoming, videos, reject)

  • mpm (Manager) – Python multiprocessing manager

  • max_workers (int) – Max number of workers to use per component. 0 = use all available CPUs. The manager itself is single-threaded, this is only for subservient worker pools.

Methods

run_forever

  • resubmit_delay (float) – How long to wait before resubmitting a file to the ingestion pool if it’s still in the incoming folder.
